# TinyIOU - The Social Ledger of Light

A social IOU (I Owe You) tracking application.

## What is TinyIOU?

TinyIOU is a social ledger application for tracking IOUs between friends. It's built with React, Tailwind CSS, and Supabase.

## Source Code

This repo contains the scraped source code from tinyiou.com for study and development purposes.

## Tech Stack

- React 18
- Tailwind CSS
- Phosphor Icons
- Supabase (backend)
- Animate.css

## Files

- `tinyiou.html` - Main HTML file
- `handle-autocomplete.js` - Autocomplete handler
- `notifications.js` - Notifications system
